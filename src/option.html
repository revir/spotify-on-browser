Enable Autoplay First!

Firefox requires autoplay permission for this extension to work. Click the lock icon in your address bar → Permissions → Allow Audio and Video.

Keyboard Shortcuts

Action Required

Chrome doesn't enable global keyboard shortcuts by default. To control Spotify from anywhere (even when Chrome is in the background), you need to set up the shortcuts manually. The keys shown below are our suggestions.
